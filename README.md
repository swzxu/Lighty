<h1 style="text-align: center;">Lighty</h1>

A user-friendly Android application with which you can control your controller for the ELK-BLEDOM LED strip and similar

## Features
* **Linking:** Connect your controller to your phone, and each time you launch the app, you will already control your LED strip.
* **Material Design:** Beautiful design created using Material 3 with Dark and White theme.
* **Speedy:** Faster than the default Chinese app


> Attention: only the ELK-BLEDOM controller has been tested, others may work, but I don't have any other controllers and therefore won't be able to do it for anothers initially.
